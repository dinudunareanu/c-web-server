About This Server

This is a basic HTTP server written in C. It supports the following features:

  • ✔️ Built using low-level POSIX sockets (socket(), bind(), listen(), accept(), etc.)
  • ✔️ Handles GET requests
  • ✔️ Serves static HTML files
  • ✔️ Serves a default index.html page for root requests
  • ✔️ Custom 404 Not Found error page
  • ✔️ Supports multiple clients at the same time using pthread
  • ✔️ Logs requests to server.log
